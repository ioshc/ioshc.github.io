1. ‘=/-‘标题 (类 Setext 形式)

Setext 1级标题

Setext 2级标题

类 Setext 形式是用底线的形式，利用 等号(=) 表示最高阶标题 和 减号(-) 表示第二阶标题
任何数量的 = 和 - 都可以有效果。 

This is an H1
=============
This is an H2
-------------

2. ‘#’标题 (类 Atx 形式)

Atx 1级标题

Atx 2级标题

Atx 3级标题

Atx 4级标题

Atx 5级标题
Atx 6级标题

类 Atx 形式则是在行首插入 1～6 个 # ，对应到标题 1～6 阶 

非闭合方式：
# 这是 H1
## 这是 H2
###### 这是 H6

闭合方式（首尾对称）：
# 这是 H1 #
## 这是 H2 ##
###### 这是 H6 ######

3. 斜体

  • 1个星号
  • 1个下划线

首尾用 1 个 星号( * ) 或者 下划线( _ ) 包含一段文本

*身正不怕影子斜*
_身正不怕影子斜_

4. 粗体

  • 2个星号
  • 2个下划线

首尾用 2 个 星号( * ) 或者 下划线( _ ) 包含一段文本

**粗不粗**
__粗不粗__

5. 粗斜体

  • 3个星号
  • 3个下划线

首尾用 3 个 星号( * ) 或者 下划线( _ ) 包含一段文本

***又粗又斜有木有***
___又粗又斜有木有___

6. 无序列表

  • 无序列表1
  • 无序列表2
  • 无序列表3

无序列表使用 星号( * ) 、加号( + ) 或是 减号( - ) 作为列表标记

*   Red
*   Green
*   Blue

等同于：
+   Red
+   Green
+   Blue

也等同于：
-   Red
-   Green
-   Blue

7. 有序列表

  1. 有序立标1
  2. 有序列表2
  3. 有序列表3

有序列表则使用数字接着一个英文句点：“1.” “2.” “3.”

1.  Bird
2.  McHale
3.  Parish

8. 链接

Markdown

格式：[链接显示的文字](URL地址)

例如：[Markdown](http://www.markdown.cn)

10. 引用（Blockquote）

普通区块引用

别人笑我太疯癫
我笑他人看不穿
不见武林豪杰墓
无花无酒锄作田 

Markdown 标记区块引用是使用类似 email 中用 > 的引用方式，

自己先断好行，然后在每行的最前面加上 > ： 
> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
> consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
> Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.
> 
> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
> id sem consectetuer libero luctus adipiscing.  

Markdown 也允许你偷懒只在整个段落的第一行最前面加上 > ：
> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
id sem consectetuer libero luctus adipiscing.
区块引用可以嵌套（引用内引用）

别人笑我太疯癫，
我笑他人看不穿，
不见武林豪杰墓，
无花无酒锄作田

你是唐伯虎？

哇呀呀呀呀~，好！实不相瞒，小弟我就是人称玉树临风胜潘安，一朵梨花压海棠的小淫虫周伯通！

区块引用可以嵌套（例如：引用内的引用），只要根据层次加上不同数量的 > ：

This is the first level of quoting.
>
> > This is nested blockquote.
>
> Back to the first level.
区块内使用其他的 Markdown 语法
我是被块引用的标题。
  1. 我是被块引用的列表1。
  2. 我是被块引用的列表2。

什么？块引用还能引用代码：

void blockquoteFunction() { printf "我就是被块引用的代码";}

引用的区块内也可以使用其他的 Markdown 语法，包括标题、列表、代码区块等：

> ## 这是一个标题。
> 
> 1.   这是第一行列表项。
> 2.   这是第二行列表项。
> 
> 给出一些例子代码：
> 
>     return shell_exec("echo $input | $markdown_script");

11. 代码

方式1

一个简单的函数：void main() { print "Hello, markdown!"; }，在同一行就可以写完

如果要标记一小段行内代码，你可以用 反引号( ` ) 把它包起来 

一个简单的函数：`void main() { print "Hello, markdown!"; }`，在同一行就可以写完

方式2

如果要标记多行代码，你可以用3个 反引号( ` ) 来开启和结束代码区段

方式3

使用 1 个制表符建立代码区块 

void main() {
    print "Hello, markdown!";
}

方式4

缩进 4 个空格建立代码区块

void main() {
    print "Hello, markdown!";
}

12. 表格

最常见的Markdown格式选项和键盘快捷键(引用自百度百科)

> 输出后的效果 Markdown 快捷键
Bold text Ctrl/⌘ + B
Emphasize text Ctrl/⌘ + I
Link [title](http://) Ctrl/⌘ + K
Inline Code code Ctrl/⌘ + Shift + K
Image ![alt](http://) Ctrl/⌘ + Shift + I
List * item Ctrl + L
Blockquote > quote Ctrl + Q
H1 # Heading
H2 ## Heading Ctrl/⌘ + H
H3 ### Heading Ctrl/⌘ + H (x2)

表格格式如下，用 >=3 个 减号( - ) 分割表头和内容，每一列用 或( | ) 分割。
默认居左，居中在分割线 ( — ) 两边加上 英文冒号( : )，居右在分割线 ( — ) 右边加上 英文冒号( : )

表头1     | 表头2           | 表头3
---      | :-------------: | ----:
默认居左   | 居中 :---:      | 居右 ---:
居左 :--- | 居中 :---:      | 居右 ---:

13. 转义符 反斜杠( \ )

Markdown支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

\   反斜杠
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  括弧
#   井号
+   加号
-   减号
.   英文句点
!   感叹号

14. 分隔线

我是一个快乐的句子，我下面有好多线。

用三个以上的 星号( * )、加号( + ) 或是 下划线( _ ) 来建立一个分隔线。 

行内不能有其他内容，你也可以在星号或是减号中间插入空格，下面每种写法都可以建立分隔线：
***
* * *
******
_ _ _
______
